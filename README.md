### About
Cinema is a Backend project that support a lot of features, with diffenent genres of movies.
### Installation

CiNeMaa requires [Node.js/Express]

#### Install the dependencies and devDependencies like:

```
npm init --yes
```
```
npm i express
```

To run the server:

```sh
nodemon server.js
```
